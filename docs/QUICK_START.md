# WriteLens - البدء السريع 🚀

## الخطوة 1️⃣: تشغيل النظام (30 ثانية فقط!)

```bash
python writelen_main.py
```

**هذا كل شيء!** ✨

النظام سيقوم بـ:
1. ✓ كشف ملف Excel تلقائيًا
2. ✓ استخراج البيانات من 7 أوراق
3. ✓ معالجتها عبر 10 مراحل AI
4. ✓ توليد 9 ملفات نتائج
5. ✓ إنشاء 3 تقارير احترافية

## أسئلة متكررة

### س: هل أحتاج إلى تثبيت مكتبات إضافية؟

**ج:** نعم، في المرة الأولى فقط:
```bash
pip install -r adaptive_writing_system/requirements.txt
```

### س: كم من الوقت تستغرق المعالجة؟

**ج:** 3-5 ثوانٍ فقط! (حسب حجم البيانات)

### س: هل يمكن معالجة طلاب متعددين؟

**ج:** نعم، تلقائيًا! إضافة أكثر من 1 طالب يعمل بدون أي تغيير

### س: أين توجد التقارير؟

**ج:** في مجلد `AI_ANALYSIS_REPORTS/` - انقر على ملف HTML لعرضه

### س: هل البيانات آمنة؟

**ج:** نعم، جميع البيانات محلية وآمنة، لا توجد عمليات تحميل

### س: ماذا إذا حدث خطأ؟

**ج:** جرّب:
```bash
python writelen_interactive.py
# ثم اختر 6 للتشخيص
```
