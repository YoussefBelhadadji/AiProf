## 🐛 استكشاف الأخطاء

### خطأ: "Cannot start full_system_launcher.py"
```bash
# تأكد أنك في المجلد الصحيح
cd C:\Users\CORTEC\Desktop\projectpr

# أو استخدم المسار الكامل
python "C:\Users\CORTEC\Desktop\projectpr\full_system_launcher.py"
```

### خطأ: "Port 5000 already in use"
```bash
# البحث عن العملية التي تستخدم المنفذ
netstat -ano | findstr :5000

# إيقافها
taskkill /PID <PID> /F
```

### خطأ: "Port 5173 already in use"
```bash
# نفس الحل أعلاه ولكن للمنفذ 5173
```
